# The Lace Wigs | Premium E-Commerce Store

A luxury, high-performance e-commerce storefront for professional-grade HD lace wigs. This project demonstrates modern front-end engineering principles, including dynamic data rendering, real-time filtering, and state management.

[LIVE DEMO LINK HERE] (e.g., https://netlify.app)

## 🚀 Key Features
- **Dynamic Product Grid:** Renders inventory from a centralized data object, mimicking a real-world API integration.
- **Advanced Filtering & Sorting:** Real-time search and category filtering with price-based sorting logic.
- **Interactive UI/UX:** Premium "Noir" aesthetic with hover-zoom effects and slide-up product details.
- **Stateful Shopping Cart:** A fully functional sliding bag that calculates totals and manages items in real-time.
- **Fully Responsive:** Mobile-first architecture ensures a seamless luxury experience on all devices.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom properties (variables), Flexbox, and CSS Grid for layout.
- **JavaScript (ES6):** Array methods (`.filter`, `.sort`, `.reduce`) for complex business logic.

## 📈 Lessons Learned
Building this project helped me master:
1. **Data Manipulation:** Handling complex arrays to provide instant feedback to user searches.
2. **Event Handling:** Efficiently managing multiple user inputs (search, dropdowns, buttons) without page reloads.
3. **Component-Based Logic:** Writing reusable JavaScript functions that bridge the gap between static HTML and dynamic apps.
